流年

2018年11月26日
| 阅读 
把酒言欢对雪吟
古今风流话语真
痴心年少轻狂梦
而立之时忽成空
流年运气一线牵
驿马行宫不得停
静默招悔钱财空
试问世间蹉跎人
谁人识得阴阳理
任运成就了此生
