Go矢量图转换

svg2data

gioui一个纯go写的UI库
git pull 之后在example里面发现了个矢量图转换工具
正如作者在文档中说的:”包iconvg为简单的矢量图形实现了紧凑的二进制格式: 图标,徽标,字形和表情符号”

这个工具有固定的命名格式ic_exampleName_48px.svg
exampleName改任意名字 数字48改为不等于1和0

之前在某灯的github上也有个类似的工具(systray) png转data 转换出来的data很大….
