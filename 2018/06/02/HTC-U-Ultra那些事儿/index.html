恢复手机短信

  • 这是个比较坑的地方 系统的“信息”不能恢复之前的备份文件
  • 下载官方刷机包：
    某度网盘:https://pan.baidu.com/s/1WjDj0OE3YdH_ShiXWyV0cA 密码: tab3
  • 解密官方刷机包:
    下载链接:https://forum.xda-developers.com/devdb/project/dl/?id=26547
    或者见度盘链接内:HTC-RUU解包工具目录下的RUU_Decrypt_Tool_3.6.5文件
    把要解包的zip文件放到软件根目录
  • 解包img文件:
    工具:度盘目录HTC-RUU解包工具目录下的Tool Unpack Repack DAT + IMG v3 x64 For Windows BY MTK THAI Developers (Thailand).7z文件 这个软件(需要python java环境)
    上面解包之后得到OUT_2PZF_1.64.1405.3这个目录
    里面有firmware和system两个目录
    firmware中都是底包
    system中含有 boot.img system.img
    把system.img放到 Tool Unpack Repack DAT + IMG v3 x64 For Windows BY MTK THAI Developers (Thailand).7Z解压后的\System_Unpack\temp_working目录下面（解压之后软件目录名字改短一点 要不然报错）
    并重命名为new.system.img
    启动软件选择1
  • 提取APP
    找到/system/priv-app/Message/ 把它扔到手机存储任意目录
    通过re文件管理器复制到系统相应的目录(/system    /priv-app/) 保持权限不变
  • 恢复信息
    找到刚才复制的目录进入安装
    先选择备份当前短信 短信备份的目录是/storage/emulated/0/sms_backup/
    把原来的备份文件放到上面目录里面点开”信息”开始恢复。。。。。。。
