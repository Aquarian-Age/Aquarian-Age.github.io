道家小六壬

三十多岁的时候接触了编程　在别人的建议下学了java
然而看了看各种的发展趋势和java死板的命名方式之后放弃了
现在回头看看 JAVA 在国内还是很吃香的　然而一切都已成过眼云烟．．．

三四年前　失业状态下学了Ｃ　买了一本书看　对我这种纯小白来说还是很吃力
后来看看视频算是基本知道了　接触了嵌入式　跟着韦东山的视频学了个基础
买了个2440也就移植了个内核进去　然后就没有然后了　这就是学完C最实战的部分了
看看内核源码　还是很多看不懂的．．．．真不知道Ｃ能让我干什么
当时还接触了Rust　就是现在币圈比较火爆的一个语言　没有一点经验看着错误抓狂　最后放弃
现在看看还是有些吃力

今年学了个Go　不得不说　Go的语法是很简洁　然而学完之后　它在我眼里就更象是一种
高级的脚本语言　虽然它号称是静态的．．．．各种的开源库都是第三方的．．．

之后接触了C# 不得不说　微软的文档是真实用　尤其是各种的Error都能在官方找到
C#语法逻辑性很好　虽然看着有些罗嗦．．．．

写了个小六壬的卜法卷　还没有写完　GitHub上蟒蛇写的那种起课方法根本就不对
常规起课方法是一宫（大安）开始沿着手指顺序排宫 后一宫位在前一宫位的基础上顺数

更新: go写了个命法卷 rust写了个卜法卷 C#不打算写了…
