入坑xfce4

系统硬件还是i3的虽然硬盘加了个SSD　然而系统的/home /opt都挂载机械上面　跑KDE实在是感觉吃力了
装了个xfce4的桌面　卸掉了gnome-keyring的一堆东西　NetworkManager-applet也跟着被卸掉了　如果装的话必须装gnome-keyring的全家通　联网的话NM的插件是用不上了　只能命令行解决一下了

更改默认桌面环境

  • 列出当前默认桌面管理
    sudo update-alternatives –list default-displaymanager

  • 更改显示管理器
    sudo update-alternatives –config default-displaymanager

    • 修改配置文件
      sudo vi /etc/sysconfig/displaymanager
      DISPLAYMANAGER=”xdm”

sudo vi /etc/sysconfig/windowmanager
DEFAULT_WM=”xfce”

配置通知服务

查看通知服务状态

systemctl –user status xfce4-notifyd

启动通知服务

验证通知是否启动

systemctl –user status xfce4-notifyd

测试通知

  • 写个脚本扔到开机启动项目里面就可以自动执行了

更新数据库

sudo updatedb && locate xfce4-notifyd.service
